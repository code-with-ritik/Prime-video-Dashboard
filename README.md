# Amazon Prime Dashboard
Developed an interactive Amazon Prime Dashboard using Power BI to analyze content distribution, genre trends, ratings, and release patterns across different years.
The dataset was cleaned and preprocessed using Excel to handle missing values, remove inconsistencies, and ensure data accuracy. Performed exploratory data analysis to identify key insights such as the most popular genres, content growth over time, and distribution of movies and TV shows.
Created multiple visualizations including bar charts, line charts, and KPI cards to represent total titles, ratings distribution, and yearly release trends.
Implemented filters and slicers to allow dynamic analysis based on genre, release year, and content type. 
The dashboard provides a clear understanding of platform content strategy and user preferences, enabling data-driven insights for better decision-making.
